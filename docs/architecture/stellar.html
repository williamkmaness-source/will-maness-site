Sable (Project Stellar)

2D puzzle-action side-scroller about a star-keeper navigating a world gone dark — concept locked, in active pre-production

Pre-production

Overview

Sable is a 2D puzzle-action side-scroller. The protagonist is a star-keeper who carries bottled starlight through a world whose stars went out — and whose civilization is failing as a result. Visual references: Hollow Knight (atmospheric world), Sky: Children of the Light (expressive silhouette characters), Ori and the Will of the Wisps (light as a living thing).

Concept and visual direction are locked. Production is underway. The project is in pre-production — world-building, character design, and the opening scene beats are established. No playable build exists yet.

This project lives in the Claude OS workstation project_stellar, separate from willmaness.com. It appears on the site as a project page (content/projects/project-stellar.mdx) but the game itself is not embedded — there is no interactive component, only editorial copy describing the project.

Key Creative Decisions

DecisionRationale
Bottled starlight as core mechanicScarcity + resource management creates tension; light = plot-relevant resource, not arbitrary
Dark is dangerous to Sable's bodyStakes are personal and immediate; each area traversal has urgency
Silhouette charactersUniversality + easier animation pipeline for solo/small team development
Hand-painted world aestheticDifferentiates from pixel art; matches "earned color" theme
