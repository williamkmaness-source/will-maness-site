Active

Overview

Viktor is a local AI stack running on a dedicated home machine also named "Viktor" — Will's personal intelligence layer that handles inbox classification, calendar summaries, newsletter extraction, morning briefings, and other automation tasks. Project Sovereign is the broader initiative encompassing Viktor and related infrastructure.

The stack is separate from willmaness.com — it does not share Vercel, Neon, or GitHub Actions. It runs on-device or via home server with Tailscale for remote access. The site project page (content/projects/local-ai-stack.mdx) contains a placeholder writeup; the full technical story is in the Claude OS workstation.

Issues for this project are tracked as markdown files in Projects/project_viktor/_issues/ — NOT as GitHub issues. This is the only project in the ecosystem that uses local markdown issue tracking.
