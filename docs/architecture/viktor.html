Overview

A personal AI agent infrastructure running on Will's local machine (named "Viktor"). The project builds a self-hosted morning briefing system: inbox triage, calendar summarization, newsletter story extraction, and daily briefing delivery via email — all orchestrated by Claude agents on a launchd schedule.

This is a separate workstation in Claude_OS, not embedded in will-maness-site. The site's content/projects/local-ai-stack.mdx provides a public-facing description.

Infrastructure Notes

  • Runs entirely on the local macOS machine — no cloud hosting
  • Google OAuth tokens require periodic renewal (see issue tracker in project_viktor/_issues/)
  • Issues tracked as markdown files in Claude_OS/Projects/project_viktor/_issues/ (not GitHub issues)
  • Tailscale MagicDNS used for local network access
