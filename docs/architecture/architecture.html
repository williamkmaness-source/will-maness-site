Overview

A complete enterprise data platform architecture and vendor evaluation deliverable set, produced as a demonstration of technical strategy and architectural thinking at the VP/Director level. The deliverable set includes reference architectures, vendor evaluation scorecards, decision frameworks, and executive-facing summaries.

All artifacts are static HTML documents stored in the Claude OS workstation at Projects/project_architecture/. The executive entry point is index.html. The deliverables are not deployed to willmaness.com but may be linked or referenced from the site.

This project is complete as of 2026-05-21. No open issues or planned extensions.

Technology Coverage (Vendor Scoring)

The vendor scorecards evaluate tools across the modern data stack:

  • Ingestion: Fivetran, Airbyte, Stitch, Hevo, dbt Cloud (source connectors)
  • Transformation: dbt Core/Cloud, SQLMesh, Coalesce
  • Compute/Warehouse: Snowflake, BigQuery, Databricks, Redshift
  • Orchestration: Airflow, Prefect, Dagster
  • Governance: Atlan, Alation, Collibra, DataHub
